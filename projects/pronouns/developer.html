Developer API

ProNouns exposes an API for Java plugin developers to use. Versioning works slightly differently for the API - the major and minor versions must match the plugin versions. For example, API v1.0 works on 1.0.0-RC1, 1.0.0, 1.0.5 etc, but not 1.5.x. Javadocs are available at https://javadoc.lucyy.me.

Currently the API is only accessible from Bukkit.

Including

Include with Maven as follows: 
<repositories>
    ...
    <repository>
        <id>lucy</id>
        <url>https://repo.lucyy.me/repository/public/</url>
    </repository>
    ...
</repositories>

<dependencies>
    ...
    <dependency>
        <groupId>me.lucyy</groupId>
        <agtifactId>pronouns-api</agtifactId>
        <version>VERSION</version>
            <scope>provided</scope>
    </dependency>
    ...
</dependencies>
Or with Gradle: 
repositories {
    maven("https://repo.lucyy.me/repository/public/")
}

dependencies {
    compileOnly("me.lucyy:pronouns-api:VERSION")
}

Getting an instance

Get an instance of PronounHandler using the RegisteredServiceProvider; 
RegisteredServiceProvider<PronounHandler> rsp = getServer()
    .getServicesManager()
    .getRegistration(PronounHandler.class);
PronounHandler handler = rsp.getProvider();
