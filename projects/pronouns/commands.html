Commands

/pronouns help - Shows all commands that the sender can use.

/pronouns set <set> [set]... - Sets the sender's pronouns.

/pronouns show [user] - Shows a player's pronouns.

  • user (optional) - the user to show pronouns for. Defaults to the sender.

/pronouns list - Lists all predefined pronoun sets.

/pronouns preview - Sends a message to the sender, substituting in their pronouns.

/pronouns version - Shows the plugin's version alongside some other information.

/pronouns reload - Reloads the config. Note that not all settings are reloaded by this command.

  • Requires the permission pronouns.admin
