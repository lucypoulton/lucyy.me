Setting your pronouns

To set your pronouns, use the "/pronouns set" command.

There are a number of different ways to set your pronouns:

Predefined Pronouns

To use these sets, you don't need to type out all 6 pronouns. Just use the first one out of these sets:
  • he
  • she
  • they
  • any (uses they/them in placeholders but shows as "Any")
  • ask (uses they/them in placeholders but shows as "Ask")
  • Server admins may add more sets to this list - use the command "/pronouns list" to see them

Custom Pronouns

If your pronouns aren't listed above, you can still use them, you just need to write them out in full. The format is as follows:

subjective/objective/progressive/possessive-adj/possessive-pronoun/reflexive

For example, the they/them pronoun set written out full would be they/them/they're/their/theirs/themself.

Multiple Sets

You can use multiple sets at the same time. If you're using multiple predefined sets, you can chain them together with a slash, for example "she/they". Otherwise, separate them with a space, for example "she they" (this works with the full set format too).
